Welcome to the DC Community Resource Inventory’s Developer Portal

The DC CRI API provides machine-readable access to the Community Resource Inventory developed by the DC Community Resource Information Exchange (DC CoRIE).
This website provides guidance and support for using the API. More information about these API protocols can be found in Open Referral’s documentation.
